## 🚀 推送代码到 GitHub

### 选项 A：在本地电脑推送（推荐）⭐⭐⭐

**优点：**
- 更安全，凭据保存在本地
- 可以在本地开发和部署
- 方便团队协作

**步骤：**

1. **在您的本地电脑打开终端**（命令提示符）

2. **克隆代码到本地**：
   ```bash
   git clone https://github.com/zzw2011341-ops/medchina.git
   cd medchina
   ```

3. **拉取工作空间的最新代码**（包含所有部署文件）：
   ```bash
   git pull https://git.coze.com/coze-coding-dev-sdk-workload/medchina.git main
   ```

4. **推送到您的 GitHub**：
   ```bash
   git push -u origin main
   ```

5. **输入认证信息**：
   - 用户名：`zzw2011341-ops`
   - 密码：**Personal Access Token**（见下方获取方法）

---

### 选项 B：获取 Token 后在这里推送⭐⭐

**如果您想让我在工作空间直接推送，请提供您的 Personal Access Token。**

**获取 Token 步骤：**

1. **访问 Token 设置页面**：
   ```
   https://github.com/settings/tokens
   ```

2. **点击**：`Generate new token` → `Generate new token (classic)`

3. **填写配置**：
   - **Note**: `MedChina Token`
   - **Expiration**: `No expiration`
   - **Scopes**: 勾选 `repo`（完整仓库权限）

4. **生成并复制 Token**：
   - 点击 `Generate token`
   - **只显示一次！立即复制！**
   - Token 格式：`ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

5. **告诉我 Token**
   - 在对话框中回复 Token
   - 我会立即帮您推送代码

## 🔑 关于 Personal Access Token

**重要提示：**

- ⚠️ **Token 不是 GitHub 登录密码**
- ⚠️ Token 只显示一次，务必立即复制保存
- ✅ Token 可以安全地用于 Git 操作
- ✅ 可以随时在 GitHub 撤销 Token

**Token 用途：**
- 推送代码到 GitHub
- 拉取代码
- 管理仓库

**安全建议：**
- 不要泄露 Token 给他人
- 不要在公共场所显示 Token
- 定期更新 Token（每 3-6 个月）
